# Hera Yacht Charter - Versão CNPJ 65.067.018/0001-56

Esta pasta contém a variação do site da **Hera Yacht Charter** focada em pacotes de fim de semana e short breaks, associada ao CNPJ **65.067.018/0001-56**.

## Visão Geral

O conteúdo destaca experiências de 1 a 3 noites a bordo, com saídas de Angra dos Reis e litoral paulista, voltadas para descanso rápido sem perder o padrão de luxo da marca.

-   **Tecnologias:** HTML5, CSS3, JavaScript (ES6+)
-   **Design:** Mesmo layout náutico e responsivo da versão base.
-   **Público-alvo:** Casais, famílias e grupos de amigos que buscam curtas escapadas.
